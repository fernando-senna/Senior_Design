Vision

The purpose of this project is to create an eye tracker system to benefit the lives of those living with disabilities. In today's market eye trackers are very expensive, they are not very accuarte, and they can be uncomfortable to wear.

Mission

Our primary goal is to re-optimize existing eye tracker algorithms and implement them in low cost and high performance MIPI cameras. The eye tracker will also be made to look modern and sleek, while being comfortable to wear for long periods of time. The eye tracker system can be made within our $800 USD budget and our future include commercially packaging it for commercialization.

Architecture

The image below gives a high level description of OptiFind by breaking it down into several layers. The entire eye tracker system consists of 3 separate layers: Software Layer, Jetson TK1, and Raspberry Pi 3. A brief description of each subsystem will be provided in the following section.

Design Details

    Software

  • Ransac algorithm used to accurately track the eye's pupil.
  • OpenCV used to analyze the image captured by the Infrared MIPI Camera
  • Other usage of OpenCV is to display the software accurately tracking the eye's pupil.
  • Capture live video stream from the Raspberry Pi 3.

    NVIDIA Jetson TK-1 Embedded Development Board

  • Development board used to implement the eye tracker algorithm.
  • Uses OpenCV4Tegra, CUDA 6.5 Toolkit, Cmake, and a g++ compiler.
  • Connects via Ethernet with the Raspberry Pi 3.
  • Infrared LEDs are powered using the Jetson TK-1 GPIO.


    Raspberry Pi 3

  • Single-board computer used to stream video via Ethernet to the Jetson TK-1.
  • An infrared MIPI Camera is connected to the Raspberry Pi 3 with an HDMI Extension Board.
  • A static IP has been set on the device for easier streaming connectivity.
  • The Raspberry Pi 3 is powered by the Jetson TK-1.
